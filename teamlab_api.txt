Пользовательские роли
● Аноним — может просматривать страницы пользователей и проектов, но кнопки взаимодействия редиректят на страницу аутентификации
● Аутентифицированный пользователь (user) — может, как и Аноним, читать всё, дополнительно он может создавать свои проекты, ставить "лайки" на другие проекты и настраивать свою страницу. Эта роль присваивается по умолчанию каждому новому пользователю.
● Администратор (admin) (Суперюзер Django)— полные права на управление всем контентом проекта. Может создавать и удалять специализации и области. Может назначать пользователей администратором.

ПОЛЬЗОВАТЕЛИ:

    СПИСОК ПОЛЬЗОВАТЕЛЕЙ:

        GET /api/v1/users/

        query parameters:
            page: integer - номер страницы
            limit: integer - количество объектов на странице
            search: string — поиск по названию скилла
            filtr: string - фильтрация по grade, status

        responses:
            200: 
                {
                "count": 123,
                "next": "http://example.ru/api/v1/users/?page=4",
                "previous": "http://example.ru/api/v1/users/?page=2",
                "results": [
                    {
                    "id": 0,
                    "username": "string",
                    "first_name": "Вася",
                    "last_name": "Иванов",
                    "grade": "middle",
                    "avatar": "http://example.ru/media/users/image.png",
                    "bio": "text",
                    "status": "available",
                    }
                ]
                }

        Поля с ограниченными значениями:

            - grade:

            junior
            middle
            senior

            - theme:

            light
            dark

            - status:

            available
            in_team
            unavailable

    РЕГИСТРАЦИЯ ПОЛЬЗОВАТЕЛЯ:

        POST /api/v1/users/

        request body schema: application/json

            email (required) <= 254 char: string (адрес электронной почты)
            username (required) <= 150 char (^[\w.@+-]+$): string (уникальный юзернейм)
            password: string (пароль)

        request:
            {
                "email": "vpupkin@yandex.ru",
                "username": "vasya.pupkin",
                "first_name": "Вася",
                "last_name": "Иванов",
                "password": "Qwerty123"
            }

        responses:
            201:

                {
                    "email": "vpupkin@yandex.ru",
                    "id": 0,
                    "username": "vasya.pupkin",
                    "first_name": "Вася",
                    "last_name": "Иванов"
                }
            
            400 (ошибка валидации):

                {
                    "field_name": [
                        "Обязательное поле."
                    ]
                }
            
                {
                    "username": ["Пользователь с таким username уже существует"]
                }
